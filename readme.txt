== Description ==

= Banckle Helpdesk Features =

**Helpdesk Automation - Invisible to Customers**
Leverage the benefits of help desk automation, without your customers having to go through any changes. Send responses from the Banckle.Helpdesk web app, get delivered as simple and personalized emails to your customers..

**Seamless Multiple Mailbox Integration**
Set up multiple support and service emails under a single online help desk account. The Banckle.Helpdesk app automatically organizes all incoming emails to their respective mailbox folders.

**Collaborative Team Inbox**
Banckle.Helpdesk becomes your collaborative Inbox, allows all team members to be on the same page with consolidated email communication. Improved productivity with online ticket system assignment and tracking features.

**Real Time Helpdesk Activity Reporting**
Stay informed with real time customer support activities. View at a glance status of all the integrated mailboxes or get detailed insights to intelligent statistics.

**Developer Friendly APIs**
Use a set of powerful web help desk APIs and build your own online helpdesk software functions right into your own applications.

**Platform Independence**
Fully compatible with Windows, Mac and Linux systems running a recent version of major web browsers. No software to download, 100% web based help desk software, run from any location at any time.


== Installation ==

= From the Archive =

1. Extract all files from the **banckle-Helpdesk** ZIP archive.
1. Upload the extracted **banckle-Helpdesk** folder to the **/wp-content/plugins/** directory.
1. Activate the plugin through the **Plugins** menu in WordPress.

= From the Wordpress admin =

1. Download the Banckle Helpdesk widget for WordPress.
1. Sign into your WordPress website admin.
1. On the left menu click *Plugins*.
	The Plugins screen displays.
1. Click *Add New*.
	The Install Plugins screen is displayed.
1. Click *Upload*.
1. Provide the path or browse to the plugin files you downloaded earlier.
1. Click *Install Now*.
	The widget is successfully installed.
1. Click *Activate Plugin*.
	The widget is successfully activated and now is ready to make it available on your website.
